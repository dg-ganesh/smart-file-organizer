# Smart File Organizer

A simple Python utility that automatically organizes files into folders based on their file type.

## Features

- Organizes files automatically
- Creates folders only when needed
- Supports common file types
- Can restore files back to the original folder
- Works with any folder containing the executable
- No user input required

## Supported File Types

| Extension | Folder |
|-----------|--------|
| .pdf | PDF |
| .doc / .docx | Word |
| .jpg / .jpeg / .png | Images |
| .ppt / .pptx | PowerPoint |
| .xls / .xlsx / .csv | Excel |
| .txt | Text |
| .zip | Zip |
| .mp4 | Videos |
| Others | Others |

## Files

- arrange.py — Organizes files
- reverse_arrange.py — Restores files back to the main folder
